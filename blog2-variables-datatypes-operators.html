01 — Theory

What is a Variable?

Think of a variable as a labelled box. You give the box a name, put something inside it, and Python remembers it for you. Whenever you need that thing again, you just use the label.

In Python, creating a variable is as simple as writing a name, an equals sign, and a value:

python
# Creating variables — name = value
city    = "Mumbai"
temp    = 34
raining = False

print(city, temp, raining)
Output
Mumbai 34 False

Notice Python didn't ask you to declare a "type" like Java or C. Python figures out the type automatically based on what you store. This is called dynamic typing — one of Python's best beginner-friendly features.

Naming rules for variables
Variable names can contain letters, numbers, and underscores. They cannot start with a number and cannot have spaces. Use snake_case (words separated by underscores) — it's the Python convention. Examples: student_name, total_score, is_active.
02 — Theory

The 4 Core Data Types

Every value in Python belongs to a data type. As a beginner, there are four types you'll use constantly:

int

Integer

Whole numbers — no decimal point. Can be positive, negative, or zero. Used for counting things.

age = 22
score = -5
year = 2024
float

Float

Numbers with a decimal point. Used for measurements, percentages, prices — anything with precision.

gpa = 8.75
price = 299.99
pi = 3.14159
str

String

Text — any characters wrapped in single or double quotes. Names, sentences, anything you'd read.

name = "Arjun"
msg = 'Hello!'
city = "Delhi"
bool

Boolean

Only two possible values: True or False. The foundation of all logic in programming.

is_student = True
has_paid = False
logged_in = True

Checking types with type()

You already saw type() in Blog 1. It tells you the data type of any value or variable. Super useful when debugging:

python
name       = "Priya"
marks      = 95
percentage = 95.0
passed     = True

print(type(name))
print(type(marks))
print(type(percentage))
print(type(passed))
Output
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>
Key insight
Notice that 95 is an int but 95.0 is a float. The decimal point makes all the difference. Python treats them differently even though they represent the same number mathematically. This will matter when you do division later.

Sometimes you need to convert a value from one type to another. Python gives you built-in functions to do this — int(), float(), str(), and bool().

python
# Converting between types
user_input = "25"          # input() always returns a string
age = int(user_input)      # convert to int to do math
print(age + 5)              # works! → 30

# str → float
price_str = "199.99"
price = float(price_str)
print(price * 1.18)         # price with 18% GST → 235.9882

# int → str (useful for printing)
score = 87
print("Your score: " + str(score))   # → Your score: 87
Output
30
235.9882
Your score: 87
Common mistake to avoid
You cannot add a string and a number directly. "Age: " + 25 will throw a TypeError. You must convert first: "Age: " + str(25). Or better — just use an f-string: f"Age: {25}". F-strings handle conversion automatically.
04 — Theory

Operators

Operators are symbols that perform operations on values. Python has several kinds — let's go through the ones you'll use every day.

Arithmetic Operators

Operator Name Example Result
+ Addition 10 + 3 13
- Subtraction 10 - 3 7
* Multiplication 10 * 3 30
/ Division (always float) 10 / 3 3.333...
// Floor division (no decimal) 10 // 3 3
% Modulo (remainder) 10 % 3 1
** Exponent (power) 2 ** 10 1024
The modulo operator — more useful than it looks
% gives you the remainder after division. This is incredibly useful in data science. For example: number % 2 == 0 tells you if a number is even. index % n is used for cycling through lists. You'll see it everywhere.

Comparison Operators

These always return True or False. They're the engine behind every if statement you'll ever write:

python
x = 10
y = 3

print(x == y)    # Equal to        → False
print(x != y)    # Not equal        → True
print(x >  y)    # Greater than     → True
print(x <  y)    # Less than        → False
print(x >= y)    # Greater or equal → True
print(x <= y)    # Less or equal    → False
Output
False
True
True
False
True
False

Assignment Operators (shortcuts)

Instead of writing x = x + 5, Python lets you write x += 5. These shorthand operators update a variable in place:

python
score = 100

score += 10     # score = score + 10  → 110
score -= 5      # score = score - 5   → 105
score *= 2      # score = score * 2   → 210
score //= 3     # score = score // 3  → 70

print(score)     # → 70
Output
70
05 — Practical

F-Strings: Python's Superpower

We touched on f-strings in Blog 1. Now let's go deeper. An f-string (formatted string literal) lets you embed any Python expression — variables, math, function calls — directly inside a string. Just prefix with f and use {curly braces}.

python
name   = "Rohan"
marks  = 450
total  = 500

# Basic variable embedding
print(f"Student: {name}")

# Math inside f-strings
print(f"Percentage: {marks/total*100}%")

# Rounding inside f-strings
print(f"Percentage: {marks/total*100:.2f}%")

# Conditional expression inside f-string
print(f"Result: {'Pass' if marks >= 250 else 'Fail'}")
Output
Student: Rohan
Percentage: 90.0%
Percentage: 90.00%
Result: Pass
The :.2f format specifier
Inside an f-string, :.2f after a variable means "show this float with exactly 2 decimal places." You'll use this constantly in data science — for displaying prices, percentages, model accuracy scores, etc. Try :.4f for 4 decimal places.
06 — Practical project

Project: Student Profile Card

Let's combine everything — variables, types, operators, and f-strings — into one practical program. This is the kind of thing you'd build in your first week at a data science internship.

student_profile.py mini project · beginner
python
# ── Student Profile Card ──────────────────────────────

# Basic info — str
name       = "Ananya Sharma"
college    = "IIT Delhi"
department = "Computer Science"

# Numbers — int and float
age          = 20
year         = 2
marks        = 432
total_marks  = 500
cgpa         = 8.9

# Booleans
scholarship  = True
hostel       = False

# Calculations
percentage   = (marks / total_marks) * 100
grad_year    = 2024 + (4 - year)

# Print the profile card
print("=" * 40)
print(f"  STUDENT PROFILE CARD")
print("=" * 40)
print(f"  Name       : {name}")
print(f"  Age        : {age}")
print(f"  College    : {college}")
print(f"  Dept       : {department}")
print(f"  Year       : {year} of 4")
print(f"  Marks      : {marks} / {total_marks}")
print(f"  Percentage : {percentage:.2f}%")
print(f"  CGPA       : {cgpa}")
print(f"  Grad Year  : {grad_year}")
print(f"  Scholarship: {'Yes' if scholarship else 'No'}")
print(f"  Hostel     : {'Yes' if hostel else 'No'}")
print("=" * 40)
Output
========================================
  STUDENT PROFILE CARD
========================================
  Name       : Ananya Sharma
  Age        : 20
  College    : IIT Delhi
  Dept       : Computer Science
  Year       : 2 of 4
  Marks      : 432 / 500
  Percentage : 86.40%
  CGPA       : 8.9
  Grad Year  : 2026
  Scholarship: Yes
  Hostel     : No
========================================

Look at how much you just did in one small program — you used all 4 data types, arithmetic operators, f-strings with formatting, and even a conditional expression inside a string. That's real Python.
