NumPy gave us the engine. Now Pandas gives us the cockpit. It's the library that turns raw data files — CSV spreadsheets, JSON exports, database dumps — into something you can inspect, interrogate, and understand in minutes.
01 — foundations

What is Pandas?

Pandas is a Python library for data manipulation and analysis. Its name comes from Panel Data — a term for multi-dimensional data sets used in statistics and econometrics.

The two core objects in Pandas are:

pd.DataFrame
A 2-dimensional table — rows and columns — like an Excel spreadsheet or a SQL table. This is what you'll work with 95% of the time. Each column is a Series.
Think: entire spreadsheet
pd.Series
A 1-dimensional labelled array — a single column or row of a DataFrame. It's like a NumPy array with an index attached to each value.
Think: one column of a spreadsheet
python
import pandas as pd
import numpy as np

# Create a DataFrame from a dictionary
data = {
    "name":   ["Arjun",  "Priya",  "Rohan",  "Sneha"],
    "age":    [22,       24,       21,       23],
    "score":  [88.5,     92.0,     75.5,     95.0],
    "passed": [True,     True,     True,     True]
}

df = pd.DataFrame(data)
print(df)

# A Series is just one column
names = pd.Series(["Arjun", "Priya", "Rohan"])
print(type(df["name"]))    # <class 'pandas.core.series.Series'>

The Universal Convention
Every data scientist writes import pandas as pd — it's as universal as import numpy as np. You'll see it in every notebook on the internet. Always use pd as the alias.
02 — loading data

Loading Data into a DataFrame

Pandas can read almost any data format. The most common by far is CSV, but you'll regularly encounter Excel files, JSON APIs, and SQL databases too.

python
# ── Reading Files ─────────────────────────────────────

# CSV — most common
df = pd.read_csv("titanic.csv")

# Excel
df = pd.read_excel("sales_data.xlsx", sheet_name="Q3")

# JSON
df = pd.read_json("users.json")

# From a URL (direct download!)
url = "https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv"
df = pd.read_csv(url)

# Useful read_csv options
df = pd.read_csv(
    "data.csv",
    sep=";",            # custom separator (semicolons)
    header=0,           # row 0 is the header
    usecols=["name", "age"],   # only load these columns
    nrows=1000,          # only first 1000 rows
    na_values=["N/A", "?", ""]  # treat these as NaN
)

04 — selecting data

Selecting Columns & Rows

Once you've loaded and understood your data, you need to access specific parts of it. Pandas gives you two ways to select: by column name, and by row index/condition.

python
# ── Column Selection ──────────────────────────────────

# Single column → returns a Series
ages = df["Age"]
print(type(ages))          # Series

# Multiple columns → returns a DataFrame
subset = df[["Name", "Age", "Survived"]]
print(subset.head())

# ── Row Selection ─────────────────────────────────────

# By index — .iloc (integer location)
print(df.iloc[0])            # first row
print(df.iloc[0:5])          # rows 0–4
print(df.iloc[0, 3])         # row 0, column 3

# By label — .loc (label-based)
print(df.loc[0, "Age"])       # row 0, "Age" column
print(df.loc[:4, ["Name", "Survived"]])

# ── Boolean / Conditional Selection ──────────────────

# Only female passengers
females = df[df["Sex"] == "female"]
print(f"Female passengers: {len(females)}")

# First class survivors
rich_survivors = df[(df["Pclass"] == 1) & (df["Survived"] == 1)]
print(f"1st class survivors: {len(rich_survivors)}")

# Passengers older than 60
elderly = df[df["Age"] > 60]
print(elderly[["Name", "Age", "Survived"]].head())
Output
Female passengers: 314
1st class survivors: 136
iloc vs loc — when to use which
.iloc uses integer positions — like list indexing. df.iloc[0] is always the first row, regardless of the index. .loc uses label-based access — it respects the actual index values. After filtering, your index might jump from 0 to 45 to 103 — .iloc[0] gives you the first remaining row, .loc[0] gives you the row where index == 0 (which might not exist). When in doubt, use .iloc.
05 — analysis methods

Essential Exploration Methods

python
# ── Value counts ──────────────────────────────────────
# How many passengers in each class?
print(df["Pclass"].value_counts())
# 3    491
# 1    216

# As percentage
print(df["Pclass"].value_counts(normalize=True).round(2))

06 — groupby

GroupBy — Aggregate by Category

groupby() is one of Pandas' most powerful features. It splits your data into groups, applies a function, and combines the results — the classic split-apply-combine pattern.

python
# Survival rate by gender
gender_survival = df.groupby("Sex")["Survived"].mean()
print(gender_survival.round(3))

# Average fare per passenger class
class_fare = df.groupby("Pclass")["Fare"].mean().round(2)
print(class_fare)

# Multiple aggregations at once
summary = df.groupby("Pclass").agg({
    "Survived": ["mean", "sum"],
    "Age":      "mean",
    "Fare":     ["mean", "max"]
}).round(2)
print(summary)
Output

The story in the numbers
Two groupby operations just revealed the entire story of the Titanic: women had a 74% survival rate vs men's 19% — "women and children first" in data form. First-class passengers paid over 6× more than third class — and had far better survival odds. This is what data exploration looks like: a few lines of code telling a human story.
07 — putting it together

Project: Full First-Look Analysis

Here's the complete workflow you'd run on any new dataset — load, inspect, explore, and extract first insights — all in one notebook cell sequence.

python
import pandas as pd

# ─ Step 1: Load ───────────────────────────────────────
url = "https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv"
df = pd.read_csv(url)
print("✓ Data loaded")

# ─ Step 2: Structure ──────────────────────────────────
print(f"\nShape   : {df.shape}")
print(f"Columns : {df.columns.tolist()}")

# ─ Step 3: Preview ────────────────────────────────────
print("\n── First 5 rows ──")
print(df.head())

# ─ Step 4: Data types & missing values ───────────────
print("\n── Data Quality ──")
print(df.info())
missing = (df.isnull().sum() / len(df) * 100).round(1)
print(missing[missing > 0])

# ─ Step 5: Statistical summary ────────────────────────
print("\n── Descriptive Stats ──")
print(df.describe())

# ─ Step 6: Key insights ───────────────────────────────
print("\n── Key Findings ──")
print(f"Survival rate      : {df['Survived'].mean()*100:.1f}%")
print(f"Average age        : {df['Age'].mean():.1f} years")
print(f"Gender split       : {df['Sex'].value_counts().to_dict()}")
print(f"Female survival    : {df[df['Sex']=='female']['Survived'].mean()*100:.1f}%")
print(f"Male survival      : {df[df['Sex']=='male']['Survived'].mean()*100:.1f}%")
print(f"1st class survival : {df[df['Pclass']==1]['Survived'].mean()*100:.1f}%")
print(f"3rd class survival : {df[df['Pclass']==3]['Survived'].mean()*100:.1f}%")
