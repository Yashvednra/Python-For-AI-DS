Theory

What exactly is Python?

Python is a high-level, interpreted programming language. Let's break that down:

  • 1
    High-level You write instructions that sound like human language. The computer handles all the low-level details (memory, hardware) behind the scenes.
  • 2
    Interpreted Python runs your code line by line, instantly. You don't need to "compile" (pre-translate) it like in C or Java. Write → Run. That's it.
  • 3
    General purpose Web apps, automation, data analysis, machine learning, scripting — Python does it all. For us, we'll focus on Data Science and AI.

What is Google Colab?

Google Colab (short for Colaboratory) is a free, browser-based coding environment. Think of it as Google Docs — but for Python code.

Why we use Colab
No installation needed. It runs on Google's servers. All major data science libraries (NumPy, Pandas, TensorFlow) come pre-installed. You just open a browser and start coding. Perfect for beginners.

A Colab notebook is made of cells. There are two types:

  • 📝
    Text cell (Markdown) For writing notes, headings, and explanations. Like this blog post itself! Written in a simple format called Markdown.
  • ▶
    Code cell Where you type and run Python code. Press Shift + Enter to run a cell. The output appears directly below it.
Practical

Let's write some code

Enough theory — let's actually code. Open colab.research.google.com and create a new notebook.

1. Your first Python program

The famous first program in every language. Type this into a code cell and press Shift + Enter:

python 
print("Hello, World!")
Output
Hello, World!

Congratulations. You just ran your first Python program. Simple, right? print() is a built-in function that displays whatever you put inside the brackets.

2. Python as a calculator

Python can do math instantly. Try these in a new code cell:

python 
# Basic arithmetic
print(10 + 5)    # Addition → 15
print(10 - 3)    # Subtraction → 7
print(4 * 6)     # Multiplication → 24
print(15 / 4)    # Division → 3.75
print(15 // 4)   # Floor division → 3 (no decimal)
print(2 ** 10)   # Power → 1024
Output
15
7
24
3.75
3
1024
What is #?
Lines starting with # are called comments. Python completely ignores them when running your code. They're just notes for you (or anyone reading your code). Always comment your code — future-you will be grateful.

3. Checking data types with type()

Every piece of data in Python has a type. The type() function tells you what type something is:

python 
print(type(42))         # int (whole number)
print(type(3.14))       # float (decimal number)
print(type("hello"))    # str (text)
print(type(True))        # bool (True or False)
Output
<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>

4. Getting help inside Python

Python has a built-in manual. Use help() to read documentation for any function, right inside Colab:

python 
help(print)   # Shows full documentation for print()
Pro tip
In Google Colab, you can also type a function name and then press Shift + Tab to get a quick popup of the documentation. Much faster than searching Google every time.

5. Put it all together — your first real program

Let's combine everything into a simple introduction program:

python 
# My introduction program

name = "Arjun"          # My name
age  = 22              # My age
goal = "Data Science"  # What I'm learning

print(f"Hello! My name is {name}.")
print(f"I am {age} years old.")
print(f"I am learning Python for {goal}.")
print(f"In 5 years I will be {age + 5} years old.")
Output
Hello! My name is Arjun.
I am 22 years old.
I am learning Python for Data Science.
In 5 years I will be 27 years old.

Notice the f"..." syntax? That's called an f-string (formatted string). It lets you embed variables directly inside text using {variable_name}. Very powerful, very readable. We'll use it constantly.
